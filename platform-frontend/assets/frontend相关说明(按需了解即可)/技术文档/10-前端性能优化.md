# 前端性能优化

## 什么是前端性能优化？

前端性能优化是指通过各种技术手段，提升网页加载速度、响应速度和用户体验的过程。

**英文原意**：Frontend Performance Optimization
**中文理解**：让网页跑得更快、更流畅、更省资源

**生活类比**：就像给汽车做保养和改装，让车子跑得更快、更省油、更稳定。

## 性能优化的重要性

### 1. 用户体验影响
- **加载时间每增加1秒**，用户流失率增加7%
- **页面响应时间超过3秒**，40%的用户会离开
- **移动端加载时间超过5秒**，74%的用户会放弃

### 2. 业务价值影响
- **亚马逊**：每100ms延迟导致1%的销售额损失
- **谷歌**：搜索结果延迟400ms，搜索量下降0.6%
- **沃尔玛**：每1秒优化，转化率提升2%

### 3. SEO影响
- **Google Core Web Vitals**：影响搜索排名
- **页面体验指标**：LCP、FID、CLS直接影响SEO

## 性能指标详解

### 1. 加载性能指标

| 指标 | 英文全称 | 中文解释 | 理想值 | 测量工具 |
|------|----------|----------|--------|----------|
| **FCP** | First Contentful Paint | 首次内容绘制 | <1.8s | Lighthouse |
| **LCP** | Largest Contentful Paint | 最大内容绘制 | <2.5s | Web Vitals |
| **TTI** | Time to Interactive | 可交互时间 | <3.8s | Lighthouse |
| **TTFB** | Time to First Byte | 首字节时间 | <600ms | WebPageTest |

### 2. 交互性能指标

| 指标 | 英文全称 | 中文解释 | 理想值 | 测量工具 |
|------|----------|----------|--------|----------|
| **FID** | First Input Delay | 首次输入延迟 | <100ms | Web Vitals |
| **CLS** | Cumulative Layout Shift | 累积布局偏移 | <0.1 | Web Vitals |
| **TBT** | Total Blocking Time | 总阻塞时间 | <200ms | Lighthouse |

### 3. 运行时性能指标

| 指标 | 英文全称 | 中文解释 | 理想值 | 测量工具 |
|------|----------|----------|--------|----------|
| **FPS** | Frames Per Second | 帧率 | 60fps | Chrome DevTools |
| **CPU** | CPU Usage | CPU使用率 | <50% | Performance |
| **Memory** | Memory Usage | 内存使用 | <200MB | Memory |

```javascript
function debounce(func, wait) {
  let timeout
  return function executedFunction(...args) {
    const later = () => {
      clearTimeout(timeout)
      func(...args)
    }
    clearTimeout(timeout)
    timeout = setTimeout(later, wait)
  }
}

function throttle(func, limit) {
  let inThrottle
  return function() {
    const args = arguments
    const context = this
    if (!inThrottle) {
      func.apply(context, args)
      inThrottle = true
      setTimeout(() => inThrottle = false, limit)
    }
  }
}
```

### 4. 缓存优化

#### HTTP缓存策略

```javascript
// 1. 强缓存（Cache-Control）
// nginx配置
location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
  expires 1y;
  add_header Cache-Control "public, immutable";
}

// 2. 协商缓存（ETag/Last-Modified）
location ~* \.(html|json)$ {
  etag on;
  last_modified on;
  add_header Cache-Control "no-cache";
}

// 3. Service Worker缓存
// sw.js
const CACHE_NAME = 'v1.0.0'
const urlsToCache = [
  '/',
  '/css/main.css',
  '/js/app.js',
  '/offline.html'
]

// 安装Service Worker
self.addEventListener('install', event => {
  event.waitUntil(
    caches.open(CACHE_NAME)
      .then(cache => cache.addAll(urlsToCache))
  )
})

// 拦截请求
self.addEventListener('fetch', event => {
  event.respondWith(
    caches.match(event.request)
      .then(response => {
        // 缓存命中，返回缓存
        if (response) {
          return response
        }
        
        // 克隆请求
        const fetchRequest = event.request.clone()
        
        return fetch(fetchRequest).then(response => {
          // 检查响应是否有效
          if (!response || response.status !== 200 || response.type !== 'basic') {
            return response
          }
          
          // 克隆响应
          const responseToCache = response.clone()
          
          caches.open(CACHE_NAME)
            .then(cache => {
              cache.put(event.request, responseToCache)
            })
          
          return response
        })
      })
      .catch(() => {
        // 离线时返回离线页面
        return caches.match('/offline.html')
      })
  )
})

// 4. IndexedDB缓存
// 使用localForage简化IndexedDB操作
import localforage from 'localforage'

// 配置
localforage.config({
  name: 'myApp',
  version: 1.0,
  storeName: 'keyvaluepairs',
  description: '应用数据缓存'
})

// 存储数据
const cacheData = async (key, data, expireTime = 3600000) => {
  const cacheItem = {
    data: data,
    expire: Date.now() + expireTime
  }
  await localforage.setItem(key, cacheItem)
}

// 获取数据
const getCachedData = async (key) => {
  const cacheItem = await localforage.getItem(key)
  
  if (!cacheItem) {
    return null
  }
  
  if (Date.now() > cacheItem.expire) {
    await localforage.removeItem(key)
    return null
  }
  
  return cacheItem.data
}

// 使用示例
const fetchDataWithCache = async (url) => {
  const cacheKey = `api_${url}`
  
  // 先检查缓存
  const cachedData = await getCachedData(cacheKey)
  if (cachedData) {
    console.log('使用缓存数据')
    return cachedData
  }
  
  // 缓存未命中，请求数据
  const response = await fetch(url)
  const data = await response.json()
  
  // 存储到缓存
  await cacheData(cacheKey, data)
  
  return data
}
```

#### 虚拟滚动

```vue
<!-- 虚拟滚动组件 -->
<template>
  <div class="virtual-list" ref="container" @scroll="handleScroll">
    <!-- 占位元素，保持滚动高度 -->
    <div class="virtual-list-phantom" :style="{ height: totalHeight + 'px' }"></div>
    
    <!-- 实际渲染的列表项 -->
    <div class="virtual-list-content" :style="{ transform: `translateY(${offsetY}px)` }">
      <div
        v-for="item in visibleData"
        :key="item.id"
        class="virtual-list-item"
        :style="{ height: itemHeight + 'px' }"
      >
        {{ item.content }}
      </div>
    </div>
  </div>
</template>

<script>
export default {
  name: 'VirtualList',
  props: {
    data: {
      type: Array,
      required: true
    },
    itemHeight: {
      type: Number,
      default: 50
    },
    visibleCount: {
      type: Number,
      default: 10
    }
  },
  data() {
    return {
      startIndex: 0,
      endIndex: 0,
      scrollTop: 0
    }
  },
  computed: {
    totalHeight() {
      return this.data.length * this.itemHeight
    },
    visibleData() {
      return this.data.slice(this.startIndex, this.endIndex)
    },
    offsetY() {
      return this.startIndex * this.itemHeight
    }
  },
  mounted() {
    this.updateVisibleRange()
  },
  methods: {
    handleScroll() {
      this.scrollTop = this.$refs.container.scrollTop
      this.updateVisibleRange()
    },
    updateVisibleRange() {
      const startIndex = Math.floor(this.scrollTop / this.itemHeight)
      const endIndex = startIndex + this.visibleCount + 2 // 多渲染2个作为缓冲
      
      this.startIndex = Math.max(0, startIndex)
```

```javascript
const optimization = {

        items: items
      }

      state.cache.set(key, {
        data,
        timestamp: Date.now()
      })
      
      // 限制缓存大小
      if (state.cache.size > 100) {
        const firstKey = state.cache.keys().next().value
        state.cache.delete(firstKey)
      }
    },
    
    // 清除过期缓存
    CLEAR_EXPIRED_CACHE(state) {
      const now = Date.now()
      const maxAge = 5 * 60 * 1000 // 5分钟
      
      for (const [key, value] of state.cache.entries()) {
        if (now - value.timestamp > maxAge) {
          state.cache.delete(key)
        }
      }
    }
  },
  
  actions: {
    // 获取大数据列表（带缓存）
    async fetchLargeDataList({ state, commit }, { page, force = false }) {
      const cacheKey = `data_list_${page}`
      
      // 检查缓存
      if (!force && state.cache.has(cacheKey)) {
        const cached = state.cache.get(cacheKey)
        if (Date.now() - cached.timestamp < 5 * 60 * 1000) {
          console.log('使用缓存数据')
          return cached.data
        }
      }
      
      // 显示加载状态
      commit('SET_LOADING', true)

        // 模拟API调用
        const response = await fetch(`/api/data?page=${page}`)
        const result = await response.json()
        
        // 更新数据
        commit('UPDATE_DATA_LIST', {

          items: result.items,
          total: result.total
        })
        
        // 缓存数据
        commit('ADD_CACHE', {
          key: cacheKey,
          data: result
        })

      } catch (error) {

      } finally {
        commit('SET_LOADING', false)
      }
    },

      }, 60 * 1000) // 每分钟清理一次

  },
```

```javascript
// 性能预算配置
const performanceBudget = {
  // 资源大小限制
  'javascript': 500 * 1024,      // 500KB
  'css': 100 * 1024,           // 100KB
  'images': 1000 * 1024,       // 1MB
  'fonts': 200 * 1024,         // 200KB
  'total': 2000 * 1024,        // 2MB
  
  // 时间限制
  'fcp': 1800,                 // 1.8s
  'lcp': 2500,                 // 2.5s
  'tti': 3800,                 // 3.8s
  'fid': 100                   // 100ms
}

// 预算检查函数
function checkPerformanceBudget(metrics) {
  const warnings = []
  
  // 检查资源大小
  for (const [type, limit] of Object.entries(performanceBudget)) {
    if (metrics[type] && metrics[type] > limit) {
      warnings.push(`${type}超出预算: ${formatBytes(metrics[type])} > ${formatBytes(limit)}`)
    }
  }
  
  // 检查时间指标
  const timeMetrics = ['fcp', 'lcp', 'tti', 'fid']
  timeMetrics.forEach(metric => {
    if (metrics[metric] && metrics[metric] > performanceBudget[metric]) {
      warnings.push(`${metric}超出预算: ${metrics[metric]}ms > ${performanceBudget[metric]}ms`)
    }
  })
  
  return warnings
}
```
