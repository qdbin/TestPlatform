## 核心功能概览

1. **AI 对话** - 基于大模型的智能问答，支持 RAG 增强和 SSE 流式输出
2. **知识库管理** - 文档的增删改查，向量检索，项目级数据隔离
3. **用例生成** - ReAct Agent 决策，自动生成符合 Schema 的测试用例
